# 2026-04-20 — Инициализация MVP

## Что сделали
- Настроили проект Next.js 14 + TypeScript + Tailwind + Supabase + RouterAI
- Написали схему БД (companies, users, documents, chunks с pgvector, clients, channels, messages, templates, integrations, action_log)
- Row Level Security изолирует данные по company_id
- Собрали лендинг с анимированным демо-чатом
- Реализовали онбординг (4 шага)
- AI-чат со streaming и RAG через pgvector
- Загрузка базы знаний (PDF/DOCX/URL/текст)
- Клиенты, документы, командный чат (Supabase Realtime)
- API-роуты для chat / upload / ingest / classify / actions
- Action Engine с tools (send_email, send_telegram, create_meeting, find_client, generate_document)
- Мобильная адаптация: 100dvh, safe-area, tab bar, font-size 16px на input

## Решения
- **Не делаем shadcn CLI** — написали минимальные UI-примитивы руками, чтобы не тянуть зависимости
- **RouterAI вместо прямых OpenAI/Anthropic** — оплата рублями, без VPN
- **pgvector IVFFlat** — дешевле и быстрее pgvector HNSW на старте (< 100k чанков)
- **Service role для embeddings/messages insert** — обходит RLS, пишем от server-side после проверки auth
