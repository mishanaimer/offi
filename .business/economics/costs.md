## Переменные (LLM)
- **RouterAI (DeepSeek V3.2)** — 25 ₽ / 1M input токенов
- **RouterAI (Claude Sonnet 4.6)** — 296 ₽ / 1M input, ~1300 ₽ / 1M output
- **Embeddings** — text-embedding-3-small, ~20 ₽ / 1M

## Себестоимость одного запроса
Средний запрос 1500 input + 500 output:
- Роутер (DeepSeek, 300 токенов): ~0.01 ₽
- Основной ответ (Sonnet, 1500 + 500): ~0.80 ₽
- **С prompt caching (45%):** ~0.45 ₽

Смешанная модель 70/20/10 → **средняя 0.56 ₽/запрос**.
