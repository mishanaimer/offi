# Roadmap — спринты

## Спринт 1 (2 нед) — готов в MVP
Auth + загрузка файлов + RAG-чат + база знаний (URL + файлы + текст).

## Спринт 2 (2 нед) — частично в MVP
Router + клиенты + командный чат (Realtime) + кастомизация ассистента.

## Спринт 3 (2 нед) — заглушки в MVP
Документы + Action Engine + Email (реальный Postmark) + Telegram + календари.

## Спринт 4 (2 нед) — не входит в MVP
AmoCRM + Bitrix24 + ЮKassa + приглашение сотрудников + публичный API.

## После MVP
- 1С (OData), Google Drive
- Голосовой ввод (Whisper через RouterAI)
- Мобильные приложения (iOS/Android через React Native или PWA)
- Общие AI-каналы (shared_ai) для всей компании
